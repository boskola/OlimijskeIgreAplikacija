# OlimijskeIgreApp

**Database**
- Relational Database (MySQL)

**Backend**
- Java
- Spring framework 
  - Spring Data JPA with Hibernate implementation
  - Dependency injection (Inversion of Control)
  - Data Transfer Object
  - Spring Security
  - Authentication and authorization
  
**Frontend**
- Javascript (ES6)
- React
   - SPA 
   - JSX
   - Sorting, Pagination
   - Bootstrap
